##반복문

크게 두가지로

* 반복적으로 실행되는 코드를 만드는 구문

* while
* for




###while
```
while(조건)
{
//구문실행
}
```

###for

```
 for(변수 생성 ; 구문 실행 조건 ; 변수 증감)
 {
 //구문 실행
 }
```
 

 
###증감연산자
```
++ ->        a++ ->        1씩 추가
-- ->        a-- ->        1씩 빼기
```
###복합연산자
```
+= ->        a +=1      a값 1씩 증가
-= ->        b -=2      b값 2씩 감소
```
